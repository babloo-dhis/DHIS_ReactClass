# Props Examples

### Definition:

Props (short for properties) are used to send a data from parent compnenet to child component.

Examples:

- Props Work like arguments passed into components.

`<Student name="Shubam" age={25}/`

### Why Propes Important?


| Reason                 | Explanation                                 |  |
| ---------------------- | ------------------------------------------- | - |
| Reusable UI            | Same Component -> different data            |  |
| Avaid code duplication | one card componenet, use 10 times and more. |  |


| Dynamic UI                    | Data changes dynamically |  |
| ----------------------------- | ------------------------ | - |
| parent -> child Communication | pass data easily         |  |
|                               |                          |  |
